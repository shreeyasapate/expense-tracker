# 💰 Expense Tracker

A modern and user-friendly Expense Tracker web application built with **Python, Flask, SQLite, HTML, CSS, JavaScript, Bootstrap, and Chart.js**. This application helps users manage their finances by tracking income, expenses, savings, and budgets through an interactive dashboard.

---

## 📌 Features

- 📊 Interactive Dashboard
- 💰 Add Income
- 💸 Add Expenses
- 📈 Financial Analytics
- 📅 Monthly Reports
- 🔍 Search & Filter Transactions
- 📝 Transaction History
- 📱 Responsive Design
- 🔐 User Authentication *(Coming Soon)*
- 🌙 Dark Mode *(Coming Soon)*

## 🚀 Installation

### Clone the repository

```bash
git clone https://github.com/YOUR_USERNAME/expense-tracker.git
```

### Go to the project folder

```bash
cd expense-tracker
```

### Create a virtual environment

```bash
python -m venv venv
```

### Activate the virtual environment

Windows

```bash
venv\Scripts\activate
```

### Install dependencies

```bash
pip install -r requirements.txt
```

### Run the application

```bash
python app.py
```

Open your browser and visit:

```
http://127.0.0.1:5000
```
